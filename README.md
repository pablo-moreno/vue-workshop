# vue-workshop

## Project setup

Create environment variables file
```
cp .env.example .env
```

Then, setup your environment variables.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
